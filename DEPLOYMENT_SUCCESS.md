## 📋 使用指南

### 1. 快速翻译
```
1. 访问 http://localhost:8000
2. 在"快速翻译"区域输入文言文
3. 点击"立即翻译"查看粤语译文
4. 点击"🔊 粤语朗读"播放语音
```

### 2. 文档翻译
```
1. 点击"选择文件"上传佛经文档
2. 支持格式：TXT、DOC、DOCX
3. 文件大小：最大100MB
4. 点击"开始翻译"进行批量翻译
5. 实时查看翻译进度和结果
```

### 3. 翻译特色
- **保留佛教术语**: 阿弥陀佛、菩萨等保持原样
- **生活化表达**: 其他内容翻译成广州大白话
- **意译方式**: 注重语境和语感
